
# Providers Movies And Series
This repository tries to share data from content providers about movies and series. for now there are only content providers for Spain. In the future we will surely add other countries (we are working on this).

From this repository you can have access to:

Json file with all the providers and contents of series and movies with the following format:

```json
{
        "imdb_score": 8.5,
        "original_title": "Avengers: Infinity War",
        "providers": {
            "buy": [
                {
                    "currency": "EUR",
                    "price": 11.99,
                    "provider": "Itunes",
                    "url": "https://itunes.apple.com/es/movie/vengadores-infinity-war/id1370714709?uo=4&at=1000l3V2"
                }
            ],
            "flatrate": [],
            "rent": []
        },
        "title": "Vengadores: Infinity War",
        "tmdb_id": 299536,
        "tmdb_score": 8.3,
        "type": "movie",
        "year": 2018
    },
```
